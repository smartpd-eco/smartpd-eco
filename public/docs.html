시스템 간편 매뉴얼

메뉴별 핵심 기능과 동작 로직 안내

입고 확인 (Scan)

바코드를 스캔하여 자재를 즉시 확인하고 수량을 기록합니다.

로직: 10초 내 취소 가능 (Undo)

재고 현황 (Inventory)

현재 보유량과 위치를 확인하고, 클릭하여 수량을 바로 수정합니다.

로직: 부족 시 자동 알림 (Red/Yellow)

처음 시작하기 (Setup)

신규 자재 등록부터 바코드 연결까지 3단계로 초기 설정을 진행합니다.

기록 보기 (History)

자재의 모든 입출고 이력을 타임라인 순으로 조회합니다.

앱 설정 (Settings)

알림 임계값 설정 및 사용자 권한을 관리합니다.

로직: 오프라인 데이터 동기화 관리
