사용 매뉴얼 & 시스템 로직

공장 자재 관리 시스템 v5를 효율적으로 사용하는 방법

사용 매뉴얼

1. 자재 입고 (BBBBB)

메인 화면의 '입고 확인' 패널에서 카메라 스캔을 통해 자재를 찾습니다. 자재가 인식되면 수량을 입력하고 '입고 기록' 버튼을 누르세요. 실수로 입력한 경우 하단에 나타나는 '취소' 버튼을 10초 이내에 클릭하여 되돌릴 수 있습니다.

2. 재고 현황 및 수정

'재고 현황' 메뉴에서는 전체 자재 목록을 볼 수 있습니다. 각 행을 클릭하면 인라인 수정 모드가 활성화되어 즉시 수량을 조정하거나 자재를 삭제할 수 있습니다.

3. 처음 시작하기 (온보딩)

새로운 자재를 등록할 때는 3단계 프로세스를 따릅니다. 자재 정보 입력 -> 바코드 스캔 -> 첫 입고 완료 순으로 진행됩니다.

핵심 시스템 로직

01
실시간 상태 연산:

재고 수량이 최소 보유량의 60% 미만이면 '위험(Red)', 90% 미만이면 '주의(Yellow)' 상태로 자동 전환됩니다.

02
낙관적 UI 업데이트:

사용자가 저장을 누르는 즉시 화면에 반영(Optimistic Update)하고, 백엔드 통신 실패 시에만 에러 토스트를 띄웁니다.

03
오프라인 우선 모드:

인터넷 연결이 끊기면 로컬 스토리지에 데이터를 임시 저장하며, 재연결 시 자동으로 동기화를 시도합니다.
